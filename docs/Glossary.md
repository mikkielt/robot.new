# Glossary

This glossary maps English terms used in documentation to their Polish counterparts used in the project context.

| English term | Polish counterpart | Meaning in plain language |
|---|---|---|
| Session | Sesja | One game event with a date, title, narrator, and outcomes. |
| Session Header | Nagłówek sesji | The session title line used as the main reference for that session. |
| Narrator | Narrator (Mistrz Gry) | The person leading or moderating a session. |
| Player | Gracz | A real person participating in the game. |
| Player Character | Postać gracza | A character controlled by a player. |
| Active Character | Aktywna postać | The character currently used by a player as their main one. |
| Entity | Encja (element świata) | Any named world element, such as a person, place, group, or player. |
| NPC | Postać niezależna (NPC) | A character not controlled by a player. |
| Organization | Organizacja | A formal or informal group/faction in the game world. |
| Location | Lokacja | A place in the game world where events or characters can be present. |
| Alias | Alias | An alternative name used for the same person, place, or group. |
| Change | Zmiana | A recorded update to world state after a session. |
| Notification | Powiadomienie | A message sent to inform someone about an update or result. |
| Webhook | Webhook (adres powiadomień) | The delivery address used for sending automated notifications. |
| PU (Player Units) | Punkty Umiejętności (PU) | Skill points assigned to characters as part of campaign progression. |
| Monthly PU Assignment | Miesięczny przydział PU | The regular process that calculates and grants PU for the selected period. |
| Starting PU | PU startowe | The baseline PU a character begins with. |
| Earned PU | PU zdobyte | PU gained by activity and progression over time. |
| Total PU | PU suma | The current overall PU value of a character. |
| Overflow PU | PU nadmiarowe | PU carried forward when monthly limits are exceeded. |
| Processed Session | Przetworzona sesja | A session already counted in operational updates, so it is not counted twice. |
| History Log | Dziennik historii | The record proving what was processed, when, and by which operation. |
| Trigger | Wyzwalacz (trigger) | A topic marker used to route relevant notifications to the right person. |
| Intel | Intel (informacja operacyjna) | Targeted in-world information sent to selected recipients. |
| Currency | Waluta | A physical in-game item representing money, tracked as a Przedmiot with a quantity (`@ilość`). |
| Quantity | Ilość | The amount of a stackable item, such as coins in a currency stack. |
| Korona (Gold) | Korona Elancka | The highest-value currency denomination. 1 Korona = 100 Talarów. |
| Talar (Silver) | Talar Hiroński | A mid-value currency denomination. 1 Talar = 100 Kogów. |
| Kog (Copper) | Kog Skeltvorski | The lowest-value currency denomination. |
